# Neon Runner

A stunning 3D cyberpunk infiltration game built with **React**, **Three.js (React Three Fiber)**, and **Node.js/Express**.

![Neon Runner Gameplay](https://via.placeholder.com/800x400?text=Neon+Runner+Gameplay+Placeholder)

## 🚀 Overview

AGENT, we have located the corrupted sector. Rogue security nodes are guarding fragments of the source code. Your mission:
1. **Infiltrate the Grid**: Navigate a high-fidelity 3D environment.
2. **Eliminate Nodes**: Use your overriding firewall logic to clear rogue security units.
3. **Recover Data**: Secure all stolen data fragments to restore the system.

## 🛠️ Tech Stack

- **Frontend**: React, Vite, React Three Fiber (Three.js), Zustand, Axios.
- **Backend**: Node.js, Express, MongoDB (Mongoose), JWT Authentication.
- **Aesthetics**: Glassmorphism, Neon Bloom effects, CRT scanline overlays.

## ⚙️ Setup & Installation

### Prerequisites
- Node.js (v18+)
- MongoDB (Running locally or Atlas)

### Backend Setup
1. `cd backend`
2. `npm install`
3. Create a `.env` file based on `.env.example` (or configure your DB URI and JWT Secret).
4. `npm run dev`

### Frontend Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## 🎮 How to Play
- **WASD / Arrows**: Move Runner.
- **Mouse**: Aim.
- **Space / Click**: Fire Decryption Pulse.
- **Goal**: Collect all green data fragments while avoiding or neutralizing red rogue nodes.
